# Talaash

`talaash` (تلاش) is an Urdu word meaning `search` or `quest.`

This tool helps researchers, students, and academics find research labs and groups that genuinely match their interests, skills, and goals.

You describe what you work on and what you are looking for. Talaash searches the web, reads lab pages, and returns a ranked list of research groups with explanations of why each one fits.
